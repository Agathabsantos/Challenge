# Challenge
Arena Fantástica

Este projeto é um jogo simples de RPG feito em Swift para rodar no console. O jogador escolhe uma classe, enfrenta inimigos aleatórios em batalhas por turnos e pode usar poções ou visitar uma loja para comprar mais itens. O objetivo do projeto é demonstrar o uso dos conceitos introdutórios de Swift aprendidos em aula, como variáveis, condicionais, coleções e funções.

Como executar:

1. Abra o Xcode e crie um projeto Command Line Tool usando Swift.

2. Abra o arquivo main.swift e substitua o conteúdo pelo código do jogo.

3. Clique em Run para iniciar o jogo no console.
